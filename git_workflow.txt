# Git Workflow for Task App Project

1. Initialize Git repository
   git init

2. Check repository status
   git status

3. Add all project files to staging
   git add .

4. Commit changes with a message
   git commit -m "Initial commit with Task App project files"

5. Connect to remote GitHub repository
   git remote add origin https://github.com/your-username/TaskApp.git

6. Push changes to remote repository (main branch)
   git push -u origin main

7. For future updates:
   git add .
   git commit -m "Updated project files"
   git push
